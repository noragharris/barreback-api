# README

Barreback-API is an API to plan an accurate Pure Barre class.

---

## Project Info

Ruby version: 2.7.2

Rails version: ~> 6.1.3

---

## Set Up

1. Install Gems

```
bundle install
```

2. Install Postgres

## Start the Server

```bash
$ bundle exec rake db:create && bundle exec rake db:reset
$ bundle exec rails s
```

## Test

```
bundle exec rspec
```
